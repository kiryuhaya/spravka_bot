# spravka_bot
бот в тг для 

1. Пользователь пишет /start
2. Бот запрашивает по очереди:
   - ФИО
   - Дата рождения (ДД.ММ.ГГГГ)
   - ИНН
   - Способ получения справки (кнопки: "Оригинал на бумаге" / "На email")
3. Если выбрано "На email" — запрашивает адрес email
4. Запрашивает фото чеков об оплате или возможность написать "Чеков нет"
5. После всего выводит сообщение:  
   "Спасибо! Справка будет оформлена в течение 30 дней."
6. Все собранные данные (ФИО, дата рождения, ИНН, способ, email если есть, статус чеков, ID пользователя, время) должны отправляться мне в личку (мой chat_id: 1660333700)
